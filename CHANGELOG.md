# Changelog

## [0.1.0] - 2025-01-01

### Added
- Initial project scaffolding
- HTML tokenizer and parser (basic HTML5 subset)
- CSS parser (core properties: color, background, margin, padding, display, font)
- DOM tree with element, text, and document nodes
- Block/inline layout engine
- LVGL-based rendering backend with canvas painting
- HTTP/1.1 client with POSIX, Win32, and stub backends
- URL parser (scheme, host, port, path, query)
- Input abstraction layer (pointer, keyboard)
- Platform abstraction layer (Linux, RTOS, bare-metal)
- Browser UI with address bar, back/forward/reload, tab bar, page view
- SDL2 desktop simulator entry point
- EoS embedded entry point
- Unit tests for HTML parser, CSS parser, DOM, layout, URL parser
- Project documentation (architecture, platform guide)
